# zapret-winws-ru
Сделан для обхода ограничений в России.

## Оригинальный софт
[zapret-win-bundle](https://github.com/bol-van/zapret-win-bundle) 

## Краткое описание файлов

* preset_general(ALT).bat - Обход иностранных сайтов + Игр.
* preset_general(ALT2).bat - Обход иностранных сайтов + Игр(Упрощенный).
* universal.bat - Основной менеджер для работы с приложением. 
 
### По умолчанию игровой фильт, фильтр айпи-адрессов отключен
